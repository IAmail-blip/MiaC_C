# Iconos PWA

Añade aquí tus iconos PNG:

- `icon-192.png` — 192×192 px (obligatorio)
- `icon-512.png` — 512×512 px (obligatorio, usado como maskable)

Recomendaciones:
- Fondo sólido (no transparente) para maskable
- Contenido centrado en el 80% del área (zona segura)
- Formato PNG con transparencia opcional para `icon-192`
